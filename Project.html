Year 1 Semester 2 Java Project

The game is a simple platformer.

As the player you have to proceed through each level whilst avoiding death. To reach the end of a level you must get to the end goal, you complete the game by reaching the end of all 5 levels.

Instructions:

Q to move Left, W to move Right. Press any mouse button to jump, you can double jump if you press twice. You can kill enemies by jumping on them (this will also reset your jump count, meaning you can jump 2 more times after killing an enemy).

Check out the code on Github: https://github.com/CitySefti/adbt136

Bootcamp 2020: Searching/Sorting Algorithms

My programming project to conclude the 2 weeks of bootcamp at city (2020). This project demonstrates understanding of arrays, loops and classes. I created a program that generates an array fill with random integers, if the user wants to they can interact by pressing 'r' to refill the array with a different set of values. Other than that, the user can press 'l' to perform a linear search, 'b' for binary search, 's' for bubble sort. I did have a second sorting algorithm, but it had problems that I could not identify - so I just decided to leave it out.

Check out the code on Github: https://github.com/CitySefti/adbt136
